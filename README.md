## Minecraft version support

### Server List Ping

| 1.7+ | 1.6 | 1.4 to 1.5 | Beta 1.8 to 1.3 |
|------|-----|------------|-----------------|
| ✓    | ✓   | ✓          | ✓               |

### Query

Query protocol is unsupported in v1.

### SRV Records

SRV Records are unsupported in v1.
